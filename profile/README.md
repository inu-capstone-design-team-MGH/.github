# Capstone_design_team

### 목표
1. 가치: 이 문제를 풀어야 할 필요가 있을까?
2. 핵심: 이 문제의 핵심요소는 무엇인가?
3. 범위: 이 문제의 어떤 부분에 집중할 것인가?
4. 현실인식: 시간 내에, 우리 능력으로 풀 수 있을까?

### 주제
경량화 기준 모델: YOLOv9-c, 모델 아키텍처 수정 및 후처리 방법을 도입하여 ==> COCO dataset으로 정확도 & 추론속도 검증
